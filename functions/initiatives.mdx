# List Function Initiatives

Retrieve a paginated list of initiatives for a specific initiative function, sorted by name.

## API Request

<Card title="GET Function Initiatives" icon="list-check">
  ```bash
  GET /api/v1/initiative-functions/{slug}/initiatives
  ```
</Card>

## Path Parameters

| Parameter | Type | Description | Required |
|-----------|------|-------------|----------|
| slug | String | The unique slug of the initiative function | Yes |

## Query Parameters

| Parameter | Type | Description | Required | Default | Range |
|-----------|------|-------------|----------|---------|-------|
| page | Integer | The page number to retrieve | No | 1 | > 0 |
| per_page | Integer | Number of items per page | No | 20 | 1 - 50 |

  Indicates if the request was successful
</ResponseField>

## Usage Notes

- Initiatives are sorted by name in ascending order
- Use pagination parameters to navigate through large result sets
- The function slug must exist in the system
- Results include detailed location and agency information for each initiative

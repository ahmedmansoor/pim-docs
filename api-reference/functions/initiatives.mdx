---
title: 'List Function Initiatives'
api: 'GET /api/v1/functions/{id}/initiatives'
---

Returns a list of all initiatives for a specific function.

<RequestExample>
  <Snippet path="/api/v1/functions/{id}/initiatives" method="get">
    ```bash {{ title: 'cURL' }}
    curl --request GET \
      --url https://pim.finance.gov.mv/api/v1/functions/123/initiatives \
      --header 'Authorization: Bearer <token>'
    ```
  </Snippet>
</RequestExample>

### Path Parameters

<ParamField path="id" type="string" required>
  The function ID
</ParamField>

### Query Parameters

<ParamField query="limit" type="integer">
  The maximum number of results to return
</ParamField>

<ParamField query="offset" type="integer">
  Number of results to skip for pagination
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array" required>
  List of initiatives for the function
</ResponseField>

<ResponseField name="meta" type="object" required>
  Pagination metadata
</ResponseField>

---
title: 'Get Function'
api: 'GET /api/v1/functions/{id}'
---

Returns details of a specific function.

<RequestExample>
  <Snippet path="/api/v1/functions/{id}" method="get">
    ```bash {{ title: 'cURL' }}
    curl --request GET \
      --url https://pim.finance.gov.mv/api/v1/functions/123 \
      --header 'Authorization: Bearer <token>'
    ```
  </Snippet>
</RequestExample>

### Path Parameters

<ParamField path="id" type="string" required>
  The function ID
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object" required>
  Function details
</ResponseField>

<ResponseExample>

```json
  "success": true,

  }
}
```
</ResponseExample>

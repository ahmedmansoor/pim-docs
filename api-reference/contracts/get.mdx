---
title: 'Get Contract'
api: 'GET /api/v1/contracts/{id}'
---

Returns details of a specific contract.

<RequestExample>
  <Snippet path="/api/v1/contracts/{id}" method="get">
    ```bash {{ title: 'cURL' }}
    curl --request GET \
      --url https://pim.finance.gov.mv/api/v1/contracts/123 \
      --header 'Authorization: Bearer <token>'
    ```
  </Snippet>
</RequestExample>

### Path Parameters

<ParamField path="id" type="string" required>
  The contract ID
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object" required>
  Contract details
</ResponseField>

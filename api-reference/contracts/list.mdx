---
title: 'List Contracts'
api: 'GET /api/v1/contracts'
---

Returns a list of all contracts.

<RequestExample>
  <Snippet path="/api/v1/contracts" method="get">
    ```bash {{ title: 'cURL' }}
    curl --request GET \
      --url https://pim.finance.gov.mv/api/v1/contracts \
      --header 'Authorization: Bearer <token>'
    ```
  </Snippet>
</RequestExample>

### Query Parameters

<ParamField query="limit" type="integer">
  The maximum number of results to return
</ParamField>

<ParamField query="offset" type="integer">
  Number of results to skip for pagination
</ParamField>

<ParamField query="initiative_id" type="string">
  Filter contracts by initiative ID
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array" required>
  List of contracts
</ResponseField>

<ResponseField name="meta" type="object" required>
  Pagination metadata
</ResponseField>

---
title: 'Get Atoll'
openapi: 'GET /api/v1/atolls/{code}'
---

Returns details of a specific atoll by its code.

### Path Parameters

<ParamField path="code" type="string" required>
  The atoll code (e.g., "HA")
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object" required>
  Atoll details
</ResponseField>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "code": "HA",
    "name": "Haa Alif Atoll",
    "name_dv": "ހއ އަތޮޅު",
    "location_id": "A00",
    "capital_island": {
      "atoll_code": "HA",
      "name": "Dhiddhoo",
      "name_dv": "ދިއްދޫ"
    }
  }
}
```
</ResponseExample>

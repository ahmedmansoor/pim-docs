# List Atoll Initiatives

Retrieve a paginated list of initiatives for a specific atoll, sorted by name.

## API Request

<Card title="GET Atoll Initiatives" icon="map-location-dot">
  ```bash
  GET /api/v1/atolls/{code}/initiatives
  ```
</Card>

## Path Parameters

| Parameter | Type | Description | Required |
|-----------|------|-------------|----------|
| code | String | The code of the atoll (e.g., "HA") | Yes |

## Query Parameters

| Parameter | Type | Description | Required | Default | Range |
|-----------|------|-------------|----------|---------|-------|
| page | Integer | The page number to retrieve | No | 1 | > 0 |
| per_page | Integer | Number of items per page | No | 20 | 1 - 50 |

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of initiative objects
  <Expandable title="Properties">
    <ResponseField name="uuid" type="string">
      Unique identifier for the initiative
    </ResponseField>
    <ResponseField name="name_english" type="string">
      Initiative name in English
    </ResponseField>
    <ResponseField name="name_dhivehi" type="string">
      Initiative name in Dhivehi
    </ResponseField>
    <ResponseField name="current_scope" type="string">
      Current initiative scope
    </ResponseField>
    <ResponseField name="location_code" type="string">
      Location code
    </ResponseField>
    <ResponseField name="location_name" type="string">
      Location name
    </ResponseField>
    <ResponseField name="status" type="object">
      Status information
      <Expandable title="Properties">
        <ResponseField name="name" type="string">Status name</ResponseField>
        <ResponseField name="slug" type="string">Status identifier</ResponseField>
        <ResponseField name="color" type="string">Status color code</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="agency" type="object">
      Agency information
      <Expandable title="Properties">
        <ResponseField name="uuid" type="string">Agency UUID</ResponseField>
        <ResponseField name="name" type="string">Agency name in English</ResponseField>
        <ResponseField name="name_dv" type="string">Agency name in Dhivehi</ResponseField>
        <ResponseField name="type" type="string">Agency type</ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Error Responses

### Atoll Not Found

```json
{
  "success": false,
  "error": {
    "code": "not_found",
    "message": "Atoll not found"
  }
}
```

## Usage Notes

- The atoll code must be a valid atoll identifier
- Initiatives are sorted by name in ascending order
- Use pagination parameters to navigate through large result sets
- Response includes detailed status and agency information for each initiative

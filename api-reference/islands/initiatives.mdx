# List Island Initiatives

Retrieve a paginated list of initiatives for a specific island, sorted by name.

## API Request

<Card title="GET Island Initiatives" icon="map-location-dot">
  ```bash
  GET /api/v1/islands/{identifier}/initiatives
  ```
</Card>

## Path Parameters

| Parameter | Type | Description | Required |
|-----------|------|-------------|----------|
| identifier | String | Either the location_id (e.g., "A01") or the island code (e.g., "LD0422") | Yes |

## Query Parameters

| Parameter | Type | Description | Required | Default | Range |
|-----------|------|-------------|----------|---------|-------|
| page | Integer | The page number to retrieve | No | 1 | > 0 |
| per_page | Integer | Number of items per page | No | 20 | 1 - 50 |

## Error Responses

### Island Not Found

```json
{
  "success": false,
  "error": {
    "code": "not_found",
    "message": "Island not found"
  }
}
```

## Usage Notes

- You can use either the location_id or island_code as the identifier
- Initiatives are sorted by name in ascending order
- Use pagination parameters to navigate through large result sets
- Response includes detailed location, status, and agency information for each initiative

## Common Use Cases

1. **Project Monitoring**: Track all development projects in a specific island
2. **Status Overview**: Get an overview of project statuses in an island
3. **Agency Activity**: Monitor which agencies are active in a particular island

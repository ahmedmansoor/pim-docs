# Get Contract Details

Get detailed information about a specific contract by its UUID.

## API Request

<Card title="GET Contract Detail" icon="file-contract">
  ```bash
  GET /api/v1/contracts/{uuid}
  ```
</Card>

## Path Parameters

| Parameter | Type | Description | Required |
|-----------|------|-------------|----------|
| uuid | String | The UUID of the contract | Yes |

## Error Responses

### Contract Not Found

```json
{
  "success": false,
  "error": {
    "code": "not_found",
    "message": "Contract not found"
  }
}
```

## Usage Notes

- The UUID must be a valid contract identifier
- Response includes detailed information about associated initiatives and project schedules
- All dates are returned in ISO 8601 format with timezone information
- Progress percentages are returned as strings to maintain precision

# List Contracts

Retrieve a paginated list of contracts with their related data.

## API Request

<Card title="GET All Contracts" icon="file-contract">
  ```bash
  GET /api/v1/contracts
  ```
</Card>

## Query Parameters

| Parameter | Type | Description | Required | Default | Possible Values |
|-----------|------|-------------|----------|---------|-----------------|
| initiative_uuid | String | Filter by initiative UUID | No | None | Valid initiative UUID |
| status | String | Filter by contract status | No | None | 'awarded', 'completed', etc. |
| page | Integer | The page number to retrieve | No | 1 | Any positive integer |
| per_page | Integer | Number of items per page | No | 20 | Min: 1, Max: 50 |

## Usage Notes

- Use pagination parameters to navigate through large sets of contracts
- Filter contracts by initiative UUID to get contracts for a specific project
- The status filter accepts status slugs (e.g., 'awarded', 'completed')
- Response includes detailed information about associated initiatives and project schedules

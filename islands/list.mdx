# List Islands

Retrieve a list of all islands with their atoll, location, and constituencies information.

## API Request

<Card title="GET All Islands" icon="map-location-dot">
  ```bash
  GET /api/v1/islands
  ```
</Card>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of island objects
  <Expandable title="Properties">
    <ResponseField name="atoll_code" type="string">
      Atoll code (e.g., "Th")
    </ResponseField>
    <ResponseField name="code" type="string">
      Island code (e.g., "LD0422")
    </ResponseField>
    <ResponseField name="name" type="string">
      Island name in English
    </ResponseField>
    <ResponseField name="name_dv" type="string">
      Island name in Dhivehi
    </ResponseField>
    <ResponseField name="location_id" type="string">
      Location identifier
    </ResponseField>
    <ResponseField name="atoll" type="object">
      Atoll information
      <Expandable title="Properties">
        <ResponseField name="code" type="string">Atoll code</ResponseField>
        <ResponseField name="name" type="string">Atoll name in English</ResponseField>
        <ResponseField name="name_dv" type="string">Atoll name in Dhivehi</ResponseField>
        <ResponseField name="location_id" type="string">Atoll location ID</ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Usage Notes

- This endpoint returns all islands without pagination
- Each island includes its associated atoll information
- Island codes follow the format "LDxxxx" where xxxx is a unique number
- Location IDs can be used to reference locations in other endpoints
- Names are provided in both English and Dhivehi when available

## Common Use Cases

1. **Location Mapping**: Use this endpoint to get a complete list of islands and their location codes for mapping purposes.
2. **Atoll Grouping**: Group islands by their atoll_code to organize them by administrative regions.
3. **Location Validation**: Validate island codes and location IDs before using them in other API requests.
